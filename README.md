## Project setup nest-backend

```bash
$ npm install
```

## Compile and run the project nest-backend

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Project setup nuxt-backend

```bash
$ npm install
```

## Compile and run the project nuxt-backend

```bash
# development
$ npm run dev

# production mode
$ npm run build

$ npm run preview
```
